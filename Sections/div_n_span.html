Div and Span elements

Div and Span are two generic container elements, we can group elements together in them, and apply different CSS designs on them to give our web-app a little more flavor.
Both of them have a class property to know which css to access and applies the visuals.

Div

The tag for div is <div> </div> Div element is a block element, all the items grouped in it will take one row block in the page.

Span

The tag for span is <span> </span> Span element is an inline element, all the items grouped in it will be in the same row with other elements, in the page.
