Blocks vs Inline elemets

Each element in HTML is considered either a block element or an inline element.
For example a paragarph <p> </p> tags are a block element, while the anchor tag <a> </a> is not.
The difference is how they are ordered on the page, blocks elements take the whole horizontal space, while inline elements grouped together in one line.
