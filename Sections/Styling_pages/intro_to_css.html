Intro to CSS

CSS (Cascading Style Sheet) is the convention to write styles for HTML pages with the idea of splitting the style from the actual html page.
Note: This page is going to remain ugly since its only purpose is to show CSS applied in different ways.
There are 3 main ways to add CSS, to see the differences on this page take a look on the source code.

  1. Inline

    Here is a paragarph with a blue background while the text is red (I know this is not nice to the eye but it's an example deal with it)

  2. Internal

    This paragarph will have a green background with a blue text using the internal way to add css

  3. External

    This is the most used way to style HTML pages, it remove the dependancy between the style to the content of the page. I added some extra here using the class attribute of the div tag with a class selector in the css file.
