HTML Boilerplate/Skeleton

Each HTML file must include 4 different tags as follow:

  1. <!DOCTYPE html> - this tag indicates that this an HTML file, there is no closing bracket for this tag.
  2. <html> </html> - this tag is the root tag all the rest of the page comes between this tag it can contain only one head tag and one body tag.
  3. <head> </head> - this tag holds the metadata of the html file.
  4. <body> </body> - this tag holds the content of the html file.

Head - The metadata

All the information stored between this tags is metadata of the page, no info from here will be shown in the actual visual of the page.

Title tag

The title tag, controls the title of the page in the metadata, the only place the title appears and is visible to the user is in two places, the first is the tab holding this window, the other one is when this page is shown as a google search result since google use this information.

Body - The content of the page

Header tag

To create a a new header we use the <h> </h> and after the h we add a number from 1 to 6, each number decides the header size, each increasing number cause a smaller header.

Text tags

To make a word bold we wrap it in the <b> <\b> tags.
To make a word italian we wrap it in the <i> <\i> tags.
To make a word have an underline we wrap it in the <u> <\u> tags.
If we want to show text a little higher or lower for example an equation with powers (x^2) we will need to use the sup and sub tags. their tags are <sup> <\sup> and <sub> <\sub> respectivly. Here is an example for pythagorian theroem to explain sup:
a2+b 2 = c 2
If we will want for example show the chemistry equation for water we will use sub, H2O

List tags

To create a list of items in the page we need one of two tags:

  • <ol></ol> which creates an ordered list.
  • <ul></ul> which creats an unordered list.

The child tags for the lists are using the tags <li></li> to declare a list item, they are used both in ordered and unordered lists the differences are that if we are doing an ordered list we will have numbers as a prefix for each list item starting at 1 and increasing as can be seen here ordered list, and inn the case of an unorderd list we will have bullet points. We can only use list items, scripts and templates in the lists tags.

Anchor tags

The Ancor tag is - <a><a> It used to add links in your text, to add a link we write href="{Some link}", then anything between the anchor tags, will be a link. The link idea works to navigate from your current page adding /{Some Link}, this can be useful to navigate inside the website, or like Wiki does with the menu section, but to actually link to the world wide web, we must add https://www. at the start of the link, so if I wanted to link to Google, and I missed the www-prefix I would get Borken Google link and the correct one will be Working Google link.

Image tags

The image tags are used to embed images inside your web. The tag for using an image is <img>, each tag must contain a source property, the path to the image file. Image of my daughter You can use more properties to setup the image, like the width and height.
